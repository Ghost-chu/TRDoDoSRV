## 配置文件

TRDoDoSRV_DoDoConfig.json （请手动放置于 `ModConfigs` 目录中）

```json
{
    "BotClientId": "机器人ClientID",
    "BotToken": "机器人Token",
    "IslandId": "你的群组 ID，机器人只会监听此群组并向此群组发送消息",
    "ChannelId": "你的频道 ID，机器人只会监听此频道并仅向此频道发送消息"
}
```
